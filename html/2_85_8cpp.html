unsigned char calculateChecksum ( const std::wstring & text)

Вычисляет контрольную сумму для заданного текста.

Эта функция вычисляет простую контрольную сумму, суммируя значения каждого символа в строке, учитывая как младший, так и старший байты каждого символа.

Parameters
textТекстовая строка, для которой вычисляется контрольная сумма.
Returns
Контрольная сумма как беззнаковый char.

Создает таблицу замены для шифра на основе заданного алфавита и текста шифра.

Эта функция генерирует отображение, которое связывает каждый символ из алфавита с соответствующим символом из текста шифра. Эта таблица используется для выполнения шифрования с помощью замены.

void encryptWithSimpleSubstitution ( const std::wstring & inputFile,
const std::wstring & outputFile )

Шифрует текстовый файл, используя выбранный шифр замены.

Эта функция считывает открытый текст из входного файла, применяет выбранный шифр с использованием шифрования простой заменой, а затем записывает зашифрованный текст и метаданные в выходной файл. Также вычисляется и записывается контрольная сумма открытого текста.

Parameters
inputFileПуть к входному текстовому файлу.
outputFileПуть к выходному файлу, в который будут записаны зашифрованный текст и метаданные.

Выбирает шифр для использования на основе текущей даты.

Шифр выбирается в зависимости от того, является ли текущий день выходным (суббота или воскресенье), и является ли месяц четным или нечетным. Это используется для выбора шифра для шифрования.

Returns
Выбранный текст шифра.
